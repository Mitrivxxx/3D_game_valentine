# 3D_game_valentine

Short 3D Valentine's browser game built to get familiar with Three.js. The project includes character movement, a simple goal system (heart vs. sad emoji), and a "hard mode" with a larger board.

## Purpose

This mini project was created to learn the basics of Three.js rendering while shipping a complete, playable prototype.

## Local setup

```bash
bun install
bun run dev:serve
```
## Tech

- Three.js
- TypeScript
- Bun

## Project scope

- Scene, lighting, and camera setup
- Map and collision logic
